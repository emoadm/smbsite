## Post-Execution Observations (from 02.3-REVIEW.md — Advisory Only)

The code review (02.3-REVIEW.md, reviewed 2026-05-10) identified 7 warnings and 8 info items. No critical findings. Key observations for follow-on work:

- **WR-01 (HIGH priority):** No `scroll-margin-top` on `<h2>` elements — headings land under the sticky header when TOC links are clicked. Fix: add `className="scroll-mt-24"` to each `<h2 id>` or set `.prose :where(h2[id]) { scroll-margin-top: 6rem; }` in globals.css.
- **WR-05 (HIGH priority):** 5 typos in chapter prose, including the brand name `Сина България` (line 739) and grammatical errors. These harm political credibility.
- **WR-02 (MEDIUM):** Duplicate `<nav aria-label="Основна навигация">` — TOC should use a distinct a11y label ("Съдържание").
- **WR-04 (MEDIUM):** 19 mixed-quote pairs — pick one convention (fully typographic or fully ASCII) and apply consistently.
- **WR-06 (LOW):** `agenda-i18n-cleanup.test.ts` uses cwd-relative path; passes from project root but fragile if run from subdirectory.
