## HTTP Method Choice

**GET** chosen over POST. Rationale:
- Idempotent semantics — cookie set is a no-op once present
- `curl -i 'http://localhost:3000/api/attr/init?utm_source=qr_letter&path=/'` reads naturally for operator smoke
- UTM params live as URL query string (mirrors how QR letter URLs are constructed)
- Browsers honor `keepalive: true` on GET fetches identically to POST
